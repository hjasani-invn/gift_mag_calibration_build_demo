This script runs test for calibration console app.

Usage:

	calib_tests.py exe_full_path target_dir
	
	exe_full_path - path to Calibration_console.exe
	target_dir - path to folder with all tests
	
Example:

	calib_tests.py C:/Github/Gift/Applications/Calibration_console/x64/Release/Calibration_console.exe //cayyc-proj01/compute02/dchurikov/temp/Robotic_survey.gimbal_test/stage7/
	
	
Output:

	Script creates 'calib_test_results.csv' file in the same folder where it is located.
